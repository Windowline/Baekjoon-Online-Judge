GoogleTestCase Class Reference

Inherits from XCTestCase
Declared in GoogleTests.mm

Overview

Base class for the generated classes for Google Test cases.

+ bundleForClass

Associates generated Google Test classes with the test bundle.

+ (NSBundle *)bundleForClass

Discussion

This affects how the generated test cases are represented in reports. By associating the generated classes with a test bundle the Google Test cases appear to be part of the same test bundle that this source file is compiled into. Without this association they appear to be part of a bundle representing the directory of an internal Xcode tool that runs the tests.
